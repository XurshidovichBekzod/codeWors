JavaScript Error Ob’ekti

Kirish

Error ob’ekti JavaScript'da xatoliklarni ifodalaydi. Bu ob’ekt yordamida xatoliklar tashlanadi (throw) va tutib olinadi (try/catch).

Sintaksis

new Error([xabar])

Asosiy Error turlari

Xatolik Turi Tavsifi
Error Umumiy xatolik
ReferenceError Mavjud bo‘lmagan o‘zgaruvchiga murojaat
TypeError Noto‘g‘ri tipdagi qiymat bilan ish
SyntaxError Sintaksisda xatolik
RangeError Qiymat oraliqdan tashqarida
URIError Noto‘g‘ri URI ishlatilgan

Misollar

1. Oddiy Error tashlash

function divide(a, b) {
  if (b === 0) {
    throw new Error("0 ga bo‘lish mumkin emas");
  }
  return a / b;
}

2. try/catch bilan tutish

try {
  divide(10, 0);
} catch (err) {
  console.error("Xatolik:", err.message);
}

3. Maxsus xatolik

class CustomError extends Error {
  constructor(message) {
    super(message);
    this.name = "CustomError";
  }
}

throw new CustomError("Bu maxsus xatolik");

4. Stack trace ko‘rish

try {
  throw new Error("Xatolik yuz berdi");
} catch (err) {
  console.log(err.stack);
}
